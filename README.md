# Sales Data Entry Project | مشروع إدخال بيانات مبيعات

### 📝 Description | الوصف
Professional sales database management project created in Microsoft Excel. It showcases data entry accuracy and organizational skills for an e-commerce electronics store.
مشروع احترافي لإدارة قاعدة بيانات مبيعات تم إنشاؤه باستخدام Microsoft Excel، يستعرض دقة إدخال البيانات ومهارات التنظيم لمتجر إلكترونيات.

### 🛠️ Skills Applied | المهارات المستخدمة
* **Data Organization:** Structured sales records with unique IDs and clear categorization.
* **Data Integrity:** Used **Data Validation** (Drop-down lists) for Order Status and Categories to prevent errors.
* **Visual Data Tracking:** Applied **Conditional Formatting** to track order status effectively.
* **Professional Formatting:** Utilized **Excel Tables** for efficient data filtering and sorting.

### 📁 Files in this Repository | الملفات المرفوعة
* **Sales_Data_Project.xlsx**: The main Excel file with all formatting and features.
